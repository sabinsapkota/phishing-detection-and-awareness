Email Phishing Detection

Paste the complete email including all headers (From, Subject, Date, etc.) for comprehensive analysis. Supports both "Show Original" format and standard email text.

In Gmail: Open email → Click 3 dots → "Show Original" → Copy all text
How to Get "Show Original" Email
Gmail
  1. Open the suspicious email
  2. Click the 3 dots (More)
  3. Select "Show Original"
  4. Copy all the text
  5. Paste in the box above
Outlook
  1. Open the email
  2. Click "..." (More actions)
  3. Select "View message source"
  4. Copy all the text
  5. Paste in the box above
Apple Mail
  1. Open the email
  2. Right-click on message
  3. Select "Show Raw Source"
  4. Copy all the text
  5. Paste in the box above
